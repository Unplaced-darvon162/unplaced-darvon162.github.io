## 🛡 Handling Installation Failures

Some users encounter errors during setup on Windows 11. Use these steps to fix common problems.

### Registry Permissions
Sometimes Windows prevents the installer from writing to the system registry. Right-click the installer file and select Run as administrator. This often bypasses access errors.

### Security Software
Antivirus programs or Windows Defender may block the installer. Check your protection history for blocked items. Allow the installer process if you identify a false positive. You should temporarily turn off real-time protection before you run the installer again.

### Compatibility Settings
1. Right-click the installer file.
2. Select Properties.
3. Click the Compatibility tab.
4. Check the box for Run this program in compatibility mode for.
5. Select Windows 10 from the list.
6. Click Apply and then try to run the installer.

### Network Adapter Drivers
NetSetMan Pro interacts with network hardware. Ensure your drivers show as updated in Device Manager. Outdated drivers cause the installation to stall or crash. Visit your laptop or motherboard manufacturer website to grab the latest network drivers.

## 💡 Using NetSetMan Pro

Once you install the software, you can manage your network profiles.

1. Open NetSetMan Pro from your desktop icon or the Start menu.
2. Select an empty profile tab on the left side of the window.
3. Input your network details such as IP address, subnet mask, and gateway.
4. Enter your DNS server settings if your work requires specific servers.
5. Click the Activate button to apply the changes to your network card.
6. Use the Auto-Switch feature to change settings based on the Wi-Fi network you connect to.

## 📋 Common Settings Reference

- DHCP: Use this if your router assigns IP addresses automatically.
- Static IP: Use this if your IT department provides specific numbers for your desk.
- Proxy: Set these if you browse the internet through an office gateway.

## 📝 Troubleshooting Summary

- If the service fails to start, restart your computer.
- If profiles do not apply, check your user privileges.
- For issues with network connection resets, ensure you save profiles before you click Activate.
- Verify that no other network management software runs in the background.
