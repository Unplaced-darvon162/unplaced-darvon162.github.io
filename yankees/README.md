# Yankees

Static Yankees section for local development and GitHub Pages.

## Run locally

Double-click `start-yankees.command`.

The launcher chooses the first open port starting at `5177`. If another local
server is already using `5177`, the site may open at `5178`, `5179`, or the next
available port.

The most recent local URL is written to `.local-url`.

## GitHub Pages

Core hosted files:

- `index.html`
- `styles.css`
- `schedule/index.html`
- `schedule/styles.css`
- `schedule/app.js`
- `team-overview/index.html`
- `team-overview/styles.css`
- `team-overview/app.js`
- `player-spotlight/index.html`
- `player-spotlight/styles.css`
- `player-spotlight/app.js`

`start-yankees.command` and `.local-url` are local development helpers.
